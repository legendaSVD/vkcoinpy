## 📋 Описание

**VKCoinPy** — это мощный и эффективный бот для автоматического майнинга виртуальной валюты VKCoin в социальной сети ВКонтакте. Бот поддерживает работу с множеством аккаунтов одновременно, автоматические переводы, умную автозакупку и множество других полезных функций.

> ⚠️ **Примечание:** Все дальнейшее описание на русском языке.

## ✨ Возможности

### 🎯 Основной функционал

- ✅ **Множественные аккаунты** - запуск нескольких ботов в одном окне
- ✅ **Высокая производительность** - можно без проблем запускать на слабых VDS и телефонах
- ✅ **Автоподключение** - автоматическое переподключение к серверу после разрыва соединения
- ✅ **Майнинг для сообщества** - возможность майнить VKCoin для группы

- 💰 **Автопереводы в процентах** - автоматический перевод определенного процента баланса

- 🎯 **Умная автозакупка** - автоматическая покупка предметов для увеличения скорости майнинга
- 📈 **Автопрокачка** - автоматическая прокачка аккаунта до определенной скорости

### 📊 Мониторинг и планирование

- 🎯 **Целевой майнинг** - возможность задать цель и узнать примерное время ее достижения
- 📈 **Отчеты о прогрессе** - регулярные отчеты о состоянии майнинга
- 🎨 **Цветной вывод** - удобное отображение информации в консоли

## 📦 Требования

### 💻 Необходимое ПО

Для запуска бота необходимо установить **Python 3.7+**.

📥 [Скачать Python](https://www.python.org/downloads/)

> ⚠️ **Важно для Windows:** Во время установки необходимо установить галочку `Add Python to Windows path`.

### 🖥 Системные требования

При запуске майнинга с **трех аккаунтов одновременно** VKCoinPy потребляет около **150 МБ оперативной памяти**. Нагрузка на процессор минимальна.

> 💡 **Таким образом, можно без проблем запускать бота на слабых VDS и даже телефонах.**

## ⚙️ Настройка

### 🔑 Получение токена

Для работы бота нужен токен ВКонтакте.

1. [Перейдите по ссылке](https://vk.cc/9f4IXA)
2. Нажмите "Разрешить"
3. Скопируйте часть адресной строки после `access_token=` и до `&expires_in`

> 💡 **Альтернатива:** Вместо токена можно использовать логин и пароль.  
> Для этого в `config.json` поставьте `vk_use_credentials: true`, укажите логин в `vk_username` и пароль в `vk_password`.

> ⚠️ **Важно:** Если у вас включена двухфакторная аутентификация - вход по логину и паролю не сработает, нужен токен.

### 📝 Настройка config.json

`config.json` может содержать несколько ботов, таким образом, можно запускать много ботов в одном окне и получать всю информацию о них.

#### Пример config.json с несколькими ботами:

```json
{
  "bots": [
    {
      "vk_token": "token1",
      "vk_use_credentials": false,
      "vk_username": "",
      "vk_password": "",
      "vk_group_id": 180735282,
      "auto_buy_enabled": false,
      "auto_buy_interval": 60.0,
      "auto_transfer_enabled": false,
      "auto_transfer_to": 0,
      "auto_transfer_when": 0,
      "auto_transfer_percent": 1,
      "goal": 1000000
    },
    {
      "vk_token": "token2",
      "vk_use_credentials": false,
      "vk_username": "",
      "vk_password": "",
      "vk_group_id": 180735282,
      "auto_buy_enabled": false,
      "auto_buy_interval": 60.0,
      "auto_transfer_enabled": false,
      "auto_transfer_to": 0,
      "auto_transfer_when": 0,
      "auto_transfer_percent": 1,
      "goal": 1000000
    }
  ],
  "log_level": "INFO"
}
```

### 📊 Параметры конфигурации

#### Параметры для каждого бота (секция `bots`):

| Параметр | Описание |
|----------|----------|
| `vk_token` | Токен страницы пользователя |
| `vk_use_credentials` | Поставить в `true`, если для входа должны быть использованы логин и пароль |
| `vk_username` | Логин ВКонтакте, если стоит `vk_use_credentials` |
| `vk_password` | Пароль ВКонтакте, если стоит `vk_use_credentials` |
| `vk_group_id` | ID сообщества, от имени которого будете майнить. Необходимо использовать `vk_use_credentials` |
| `auto_buy_enabled` | Автозакупка (`true` - включена, `false` - выключена) |
| `auto_buy_interval` | Интервал автозакупки в секундах |
| `auto_buy_target_tick` | Скорость, до которой будет производиться автозакупка (например, 20 коинов\тик = 20) |
| `auto_transfer_enabled` | Автоперевод (`true` - включен, `false` - выключен) |
| `auto_transfer_to` | ID пользователя для автоперевода |
| `auto_transfer_when` | По достижению какой суммы выполнять перевод |
| `auto_transfer_percent` | Сколько процентов баланса переводить |
| `auto_transfer_interval` | Интервал автоперевода |
| `goal` | Цель в коинах |
| `progress_report_enabled` | Включить вывод сообщений о прогрессе для каждого бота |
| `progress_report_interval` | Интервал сообщений о прогрессе в секундах |

#### Общие настройки для всех ботов:

| Параметр | Описание |
|----------|----------|
| `report_enabled` | Отчет по всем ботам вкл\выкл |
| `report_interval` | Интервал отчета по всем ботам в секундах |
| `log_level` | Уровень логирования (INFO, DEBUG, ERROR) |

## 🏢 Майнинг для сообщества

Для того, чтобы майнить VKCoin для сообщества, необходимо указать `vk_group_id` (ID сообщества).

> ⚠️ **Важно:** Майнинг для сообщества работает только в том случае, если вы указали `vk_use_credentials: true`, и правильно заполнили `vk_username`, `vk_password`.

## ❓ Часто задаваемые вопросы

### ❓ Появляется ошибка `No translation file found for domain...`, что делать?

Ошибка может возникать на Windows. В последних версиях бота она полностью исправлена.  
**Решение:** Обновить бота, либо запускать его не через `start.bat`, а из командной строки.

### ❓ Как включить умную автопокупку?

Умная автопокупка **включена по умолчанию**. Просто установите `auto_buy_enabled: true` в конфигурации.

### ❓ Можно ли запускать несколько ботов одновременно?

Да! Просто добавьте несколько записей в секцию `bots` в `config.json`. Все боты будут работать параллельно в одном окне.

### ❓ Как узнать ID сообщества?

1. Перейдите на страницу сообщества
2. В адресной строке вы увидите что-то вроде `vk.com/club123456789`
3. Число после `club` и есть ID сообщества (или используйте число после `public`)
